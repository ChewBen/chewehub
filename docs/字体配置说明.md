# 字体配置说明

## 已配置的字体

项目已配置以下自定义字体，可在 Tailwind CSS 中直接使用：

### 1. Mulish 可变字体
- **文件**: `Mulish-VariableFont_wght.ttf`, `Mulish-Italic-VariableFont_wght.ttf`
- **特点**: 可变字体，支持 100-900 字重
- **Tailwind 类**: `font-mulish`
- **使用场景**: 英文正文

### 2. Dream Han Sans W14 (思源黑体 W14)
- **文件**: `DreamHanSans-W14.ttc`
- **特点**: 中文字体，宽度为 W14
- **Tailwind 类**: `font-dream-sans-w14`
- **使用场景**: 中文正文（窄版）

### 3. Dream Han Sans W20 (思源黑体 W20)
- **文件**: `DreamHanSans-W20.ttc`
- **特点**: 中文字体，宽度为 W20
- **Tailwind 类**: `font-dream-sans-w20`
- **使用场景**: 中文正文（宽版）

### 4. Dream Han Serif W20 (思源宋体 W20)
- **文件**: `DreamHanSerif-W20.ttc`
- **特点**: 中文衬线字体
- **Tailwind 类**: `font-dream-serif-w20`
- **使用场景**: 中文标题

### 5. 混合字体家族

#### mix-sans（混合无衬线字体）
- **Tailwind 类**: `font-mix-sans`
- **字体顺序**: Mulish → Dream Han Sans W20
- **使用场景**: 中英文混排正文（推荐）
- **注意**: 仅使用配置的自定义字体，无备用字体

#### mix-serif（混合衬线字体）
- **Tailwind 类**: `font-mix-serif`
- **字体顺序**: Dream Han Serif W20
- **使用场景**: 中英文混排标题（推荐）
- **注意**: 仅使用配置的自定义字体，无备用字体

### 6. 默认字体配置

项目已配置默认字体，覆盖 Tailwind 的默认字体设置：
- **默认无衬线字体 (sans)**: Mulish → Dream Han Sans W20
- **默认衬线字体 (serif)**: Dream Han Serif W20
- **默认等宽字体 (mono)**: Mulish

**重要**: 所有字体配置**仅使用自定义字体**，**不包含任何备用字体或系统字体**。

## 使用方法

### 在 Tailwind CSS 中使用

```vue
<template>
  <!-- 使用 Mulish 字体 -->
  <p class="font-mulish">Hello World</p>
  
  <!-- 使用 Dream Han Sans W20 -->
  <p class="font-dream-sans-w20">中文内容</p>
  
  <!-- 使用混合字体（推荐） -->
  <p class="font-mix-sans">中英文混排 Mixed Content</p>
  
  <!-- 使用混合衬线字体（标题） -->
  <h1 class="font-mix-serif">标题 Title</h1>
</template>
```

### 在 SCSS/CSS 中使用

```scss
/* 注意：所有字体配置仅使用自定义字体，不包含备用字体 */

.my-class {
  font-family: 'Mulish';  /* 仅使用 Mulish，无备用字体 */
}

.chinese-text {
  font-family: 'Dream Han Sans W20';  /* 仅使用自定义字体 */
}

.mixed-content {
  font-family: 'Mulish', 'Dream Han Sans W20';  /* 仅使用这两种字体 */
}
```

## 字体文件位置

所有字体文件位于：`cheweHub-ui/src/assets/fonts/`

## 配置文件

- **字体定义**: `cheweHub-ui/src/assets/styles/fonts.css`
- **Tailwind 配置**: `cheweHub-ui/tailwind.config.js`
- **样式引入**: `cheweHub-ui/src/assets/styles/tailwind.css`

## 注意事项

1. **仅使用自定义字体**: 项目配置**仅使用**上述 5 种自定义字体，**不包含任何备用字体或系统字体**
2. **TTC 文件支持**: Dream Han 字体使用 TTC (TrueType Collection) 格式，现代浏览器支持良好
3. **字体加载**: 使用 `font-display: swap` 确保文字始终可见
4. **性能考虑**: 字体文件较大（总计约 90MB），建议确保字体文件能正常加载
5. **字体回退**: **无备用字体**，如果自定义字体加载失败，浏览器将使用默认字体（通常为浏览器默认字体）

## 推荐使用方案

### 方案一：混合字体（推荐）
```vue
<div class="font-mix-sans">
  <p>正文内容</p>
</div>

<h1 class="font-mix-serif">标题</h1>
```

### 方案二：分别指定
```vue
<div class="font-mulish">
  <p>English Content</p>
</div>

<div class="font-dream-sans-w20">
  <p>中文内容</p>
</div>
```

## 字体预览

- **Mulish**: 现代、简洁的无衬线字体，适合英文内容
- **Dream Han Sans**: 优雅的中文无衬线字体，适合正文
- **Dream Han Serif**: 传统的中文衬线字体，适合标题和强调
