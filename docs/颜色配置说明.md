# Typography 颜色配置说明

## 统一的颜色配置

所有 Typography 风格页面的颜色已统一配置，便于管理和维护。

## 配置文件位置

### 1. Tailwind CSS 配置
- **文件**: `cheweHub-ui/tailwind.config.js`
- **配置项**: `theme.extend.colors.typography`

### 2. SCSS 颜色变量
- **文件**: `cheweHub-ui/src/assets/styles/typography-colors.scss`
- **变量名**: `$typography-bg`, `$typography-fg` 等

### 3. CSS 变量
- **文件**: `cheweHub-ui/src/assets/styles/typography-colors.css`
- **变量名**: `--typography-bg`, `--typography-fg` 等

## 颜色定义

### 基础颜色

```javascript
// Tailwind 配置 (tailwind.config.js)
colors: {
  'typography': {
    'bg': '#ffffff',           // 背景色
    'fg': '#2e405b',           // 前景色（文字颜色）
    'fg-dark': '#1a2d42',      // 深色前景（focus 状态）
    'fg-light': '#3d5670',     // 浅色前景（hover 状态）
  },
}
```

```scss
// SCSS 变量 (typography-colors.scss)
$typography-bg: #ffffff;           // 背景色
$typography-fg: #2e405b;           // 前景色
$typography-fg-dark: #1a2d42;      // 深色前景（darken 30%）
$typography-fg-light: #3d5670;     // 浅色前景（lighten 10%）
```

```css
/* CSS 变量 (typography-colors.css) */
:root {
  --typography-bg: #ffffff;
  --typography-fg: #2e405b;
  --typography-fg-dark: #1a2d42;
  --typography-fg-light: #3d5670;
}
```

## 使用方法

### 1. 在 Tailwind CSS 中使用

```vue
<template>
  <!-- 使用 Tailwind 颜色类 -->
  <div class="bg-typography-bg text-typography-fg">
    <p class="text-typography-fg">文字内容</p>
    <a class="hover:text-typography-fg-light focus:text-typography-fg-dark">链接</a>
  </div>
</template>
```

### 2. 在 SCSS 中使用

```scss
// 引入颜色配置
@import '@/assets/styles/typography-colors.scss';

.my-class {
  background-color: $typography-bg;
  color: $typography-fg;
  
  &:hover {
    color: $typography-fg-light;
  }
  
  &:focus {
    color: $typography-fg-dark;
  }
}
```

### 3. 在 CSS 中使用

```css
.my-class {
  background-color: var(--typography-bg);
  color: var(--typography-fg);
}

.my-class:hover {
  color: var(--typography-fg-light);
}

.my-class:focus {
  color: var(--typography-fg-dark);
}
```

### 4. 在 JavaScript 中使用

```javascript
// 使用 CSS 变量
const bgColor = getComputedStyle(document.documentElement)
  .getPropertyValue('--typography-bg');
const fgColor = getComputedStyle(document.documentElement)
  .getPropertyValue('--typography-fg');
```

## 页面应用示例

论坛页面 (`/forum/index`) 已使用统一颜色配置：

```vue
<style lang="scss" scoped>
// 引入统一的颜色配置
@import '@/assets/styles/typography-colors.scss';

// 使用统一配置的颜色
$backgroundColor: $typography-bg;
$foregroundColor: $typography-fg;
$foregroundColorDark: $typography-fg-dark;
$foregroundColorLight: $typography-fg-light;
</style>
```

## 颜色使用场景

| 颜色 | 使用场景 | Tailwind 类 | SCSS 变量 |
|------|---------|------------|----------|
| `bg` (#ffffff) | 页面背景 | `bg-typography-bg` | `$typography-bg` |
| `fg` (#2e405b) | 主要文字颜色、边框 | `text-typography-fg` | `$typography-fg` |
| `fg-dark` (#1a2d42) | Focus 状态、强调 | `text-typography-fg-dark` | `$typography-fg-dark` |
| `fg-light` (#3d5670) | Hover 状态 | `text-typography-fg-light` | `$typography-fg-light` |

## 修改颜色

如需修改颜色，请按以下步骤：

1. **修改 Tailwind 配置** (`tailwind.config.js`)
2. **修改 SCSS 变量** (`typography-colors.scss`)
3. **修改 CSS 变量** (`typography-colors.css`)

确保三个文件中的颜色值保持一致！

## 最佳实践

1. **优先使用 Tailwind 类**：`bg-typography-bg`, `text-typography-fg`
2. **SCSS 中使用变量**：`$typography-bg`, `$typography-fg`
3. **避免硬编码**：不要直接写 `#ffffff` 或 `#2e405b`
4. **保持一致性**：所有页面使用相同的颜色配置

## 注意事项

- 所有颜色值已统一配置，修改时请同步更新所有配置文件
- 颜色配置文件已自动引入，可直接使用
- 支持 Tailwind CSS、SCSS 变量、CSS 变量三种方式
