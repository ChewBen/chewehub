```javascript
export default {
  content: [
    "./index.html",
    "./src/**/*.{vue,js,ts,jsx,tsx}",
  ],
  theme: {
    extend: {
      colors: {
        'typography': {
          'bg': '#ffffff',
          'fg': '#2e405b',
          'fg-dark': '#1a2d42',
          'fg-light': '#3d5670',
        },
      },
      fontFamily: {
        'mulish': ['Mulish'],
        'dream-sans-w14': ['Dream Han Sans W14'],
        'dream-sans-w20': ['Dream Han Sans W20'],
        'dream-serif-w20': ['Dream Han Serif W20'],
        'mix-sans': ['Mulish', 'Dream Han Sans W20'],
        'mix-serif': ['Dream Han Serif W20'],
        'sans': ['Mulish', 'Dream Han Sans W20'],
        'serif': ['Dream Han Serif W20'],
        'mono': ['Mulish'],
      },
    },
  },
  plugins: [],
  corePlugins: {
    preflight: true,
  },
}
```

### vite.config.js (关键部分)

```javascript
import tailwindcss from '@tailwindcss/vite'

export default defineConfig({
  plugins: [
    vue(),
    tailwindcss({
      config: './tailwind.config.js'
    }),
  ],
  resolve: {
    alias: {
      '@': path.resolve(__dirname, './src')
    }
  },
  server: {
    port: 39090,
    proxy: {
      '/dev-api': {
        target: 'http://localhost:38080',
        changeOrigin: true,
        rewrite: (p) => p.replace(/^\/dev-api/, '')
      }
    }
  }
})
```

### src/assets/styles/tailwind.css



## 请求和鉴权配置

### 必须创建的文件

```
src/utils/
├── request.js          # Axios 请求配置（包含拦截器和鉴权）
├── auth.js             # Token 管理工具
└── errorCode.js        # 错误码映射

src/plugins/
└── cache.js            # 缓存插件（session/localStorage）

src/permission.js       # 路由守卫配置
```

### 请求配置要点

**Token存储：**
- 使用 `js-cookie` 存储在 Cookies
- Token键名：`Admin-Token`
- 请求头格式：`Authorization: Bearer {token}`

**请求特性：**
- 默认 Content-Type: `application/json;charset=utf-8`
- 超时时间：10秒
- 防重复提交：POST/PUT 请求默认启用（1秒间隔）
- GET 参数自动转换为 URL 查询字符串

**响应处理：**
- 401：自动跳转登录页
- 500：显示错误消息
- 601：显示警告消息
- 其他非200：显示错误通知

### 路由守卫配置

**白名单路由：**
- `/login` - 登录页
- `/register` - 注册页

**守卫逻辑：**
- 有Token + 访问登录页 → 重定向首页
- 无Token + 访问受保护页面 → 重定向登录页（带redirect）
- 有Token + 无用户信息 → 获取用户信息并生成动态路由
