# 字体统一配置总结

## 已完成的工作

### 1. 移除不需要的字体

- ✅ 移除了 Google Fonts 的引入（Source Sans Pro, Source Serif Pro）
- ✅ 移除了旧的字体变量定义（`$fontList`, `$titleFontList`）
- ✅ 移除了系统字体备用列表（Helvetica, Roboto, PingFang 等）

### 2. 统一使用 Tailwind 配置的字体

所有字体现在都通过 Tailwind CSS 配置统一管理，使用 Tailwind 类应用：

**配置文件：**
- `cheweHub-ui/tailwind.config.js` - 字体配置
- `cheweHub-ui/src/assets/styles/fonts.css` - 字体文件定义

**可用的字体类：**
- `font-sans` - 混合中英文字体（Mulish + Dream Han Sans W20）
- `font-serif` - 衬线字体（Dream Han Serif W20）
- `font-mulish` - 仅 Mulish 字体
- `font-dream-sans-w14` - Dream Han Sans W14
- `font-dream-sans-w20` - Dream Han Sans W20
- `font-dream-serif-w20` - Dream Han Serif W20
- `font-mix-sans` - 混合字体（Mulish + Dream Han Sans W20）
- `font-mix-serif` - 混合衬线字体（Dream Han Serif W20）

### 3. 页面字体应用

在 `/forum/index` 页面中：

```vue
<template>
  <!-- 主容器：使用默认混合字体 -->
  <div class="typography-blog-container font-sans">
    
    <!-- 侧边栏：使用 Mulish 字体 -->
    <aside class="side-container font-mulish">
      
      <!-- 标题区域：使用衬线字体 -->
      <div class="vertical-text site-title font-serif">
        ...
      </div>
    </aside>
    
    <!-- 主内容区：使用默认混合字体 -->
    <main class="main-container font-sans">
      ...
    </main>
  </div>
</template>
```

### 4. 字体配置详情

**Tailwind 配置 (`tailwind.config.js`)：**

```javascript
fontFamily: {
  'mulish': ['Mulish'],
  'dream-sans-w14': ['Dream Han Sans W14'],
  'dream-sans-w20': ['Dream Han Sans W20'],
  'dream-serif-w20': ['Dream Han Serif W20'],
  'mix-sans': ['Mulish', 'Dream Han Sans W20'],
  'mix-serif': ['Dream Han Serif W20'],
  'sans': ['Mulish', 'Dream Han Sans W20'],  // 默认字体
  'serif': ['Dream Han Serif W20'],
  'mono': ['Mulish'],
}
```

**字体文件位置：**
- `cheweHub-ui/src/assets/fonts/` - 所有字体文件
- `cheweHub-ui/src/assets/styles/fonts.css` - `@font-face` 定义

### 5. 优势

1. **统一管理**：所有字体在 Tailwind 配置中集中管理
2. **性能优化**：移除了外部字体加载（Google Fonts）
3. **灵活性**：通过 Tailwind 类轻松切换字体
4. **一致性**：所有页面使用相同的字体配置
5. **可维护性**：修改字体只需更新配置文件

### 6. 使用指南

#### 在模板中使用字体类

```vue
<!-- 使用默认混合字体 -->
<div class="font-sans">正文内容</div>

<!-- 使用衬线字体（标题） -->
<h1 class="font-serif">标题</h1>

<!-- 使用特定字体 -->
<div class="font-mulish">Mulish 字体</div>
```

#### 在 SCSS 中使用字体

```scss
// 不推荐：直接定义字体
// font-family: 'Mulish', 'Dream Han Sans W20';

// 推荐：使用 Tailwind 类
// 在模板中使用 class="font-sans"
```

### 7. 注意事项

- ✅ 所有字体已移除系统字体备用，只使用配置的字体
- ✅ 不再使用 Google Fonts，减少外部依赖
- ✅ 字体通过 Tailwind 类应用，不在 SCSS 中硬编码
- ✅ 颜色和字体配置已完全分离，便于独立管理

1. **其他页面**：将其他页面的字体也统一到 Tailwind 配置
2. **字体优化**：根据需要调整字体权重和样式
3. **性能监控**：监控字体加载性能，必要时进行优化
